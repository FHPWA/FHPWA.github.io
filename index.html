BlackC4t OTP Version 1.3

BlackC4t OTP Version 1.3 (Web Edition) uses the One Time Pad (OTP) encryption method. This requires the use of a pre-shared key to encrypt plain text. This is a form of symmetric encryption and is therefore not recommended to be used as a primary form of encryption. As of version 1.1 JavaScript has been used instead of Python to improve the page speed.
